TOKYO ARCADE

Sistema de catálogo de videojuegos desarrollado en ASP.NET Core MVC. Esta plataforma permite la gestión, visualización y reseña de una colección de videojuegos con una interfaz personalizada.
Funcionalidades principales

    Búsqueda y filtrado: Implementación de búsqueda por título y filtrado de elementos por categoría desde la vista principal.

    Sistema de reseñas: Funcionalidad para que los usuarios publiquen opiniones y calificaciones con persistencia local. Permite la edición y eliminación de reseñas.

    Interfaz de usuario: Diseño estilizado con elementos interactivos, efectos visuales de navegación y estructura de información optimizada.

    Fichas técnicas: Visualización de detalles de cada juego mediante una estructura de datos organizada.


Uso de IA: Use la ia en arreglar el codigo y dejar bonito el codigo y la pagina
